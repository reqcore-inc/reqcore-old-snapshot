# Reqcore

## Archived open-source release

This repository is an archive of the final open-source, self-hosted version of Reqcore.

Reqcore is no longer maintained as an open-source project. This repository is provided as-is for existing users and for anyone who wants to inspect or self-host the last open-source release. Newer development happens outside this repository and is not published here.

## Maintenance status

- This repository is archived and no longer actively maintained.
- Issues and pull requests may not be reviewed.
- Security fixes, dependency updates, compatibility updates, and new features should not be expected.
- Use this code at your own discretion, especially for production deployments.

Reqcore was an open-source applicant tracking system for self-hosted hiring workflows. This archive preserves the last open-source code snapshot for users who relied on that release.
